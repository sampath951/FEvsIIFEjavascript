Difference between FE vs IIFE
---------------------------------------------------------------------------------
Function Expression(FE) can call function many times as we want
Function Expression(FE) need not to call function after immediate defining them
FE is useful for code reusability and only once used in program.
----------------------------------------------------------------------------------
Immediate Invoke Function Expression(IIFE) can call functions only once
Immediate Invoke Function Expression(IIFE) call function immediate defining them
IIFE is not useful for code reusability but useful for big programs in sub IIFE like IIFE1,IIFE2,-- so on..
----------------------------------------------------------------------------------
